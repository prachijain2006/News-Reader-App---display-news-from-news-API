# OP-Stories: News Reader App

OP-Stories is a Flutter-based mobile news reader application that provides users with real-time news updates from multiple categories using the News API. The app is designed with a clean and responsive user interface, enabling users to browse headlines, view detailed articles, and open full news stories in an external browser.

## Features

* Real-time news fetching using News API
* Categorized news display for easy navigation
* Responsive and user-friendly Flutter UI
* Detailed article view with title, description, image, author, source, and publication date
* External browser integration using `url_launcher`
* Error handling for network issues and invalid API responses

## Technologies Used

* Flutter SDK
* Dart
* News API
* HTTP package
* url_launcher package
* Android Studio

## Project Objective

The main objective of OP-Stories is to provide a lightweight, fast, and user-friendly platform for accessing real-time news updates without unnecessary complexity. The project demonstrates practical implementation of Flutter app development, API integration, JSON data handling, asynchronous programming, and UI/UX design.
